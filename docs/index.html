Features

Everything in your menu bar

📡

Real-time Monitoring

Automatically detects and tracks all running Claude Code sessions: CLI, Xcode, Claude Desktop Code, and Claude Desktop Cowork.

🤖

Subagent Grouping

Sessions spawned via the Agent tool (e.g. SO-ADK pipelines) are folded under their parent with a 🤖×N badge. Click to expand and see each subagent's type, task, and last response on hover. Parent token/cost totals roll up every subagent so you see full pipeline cost at a glance.

🏷️

Source Badges

Each session is labeled by origin (Code, Cowork, Xcode) so you always know where it's running. AI-generated session titles replace cryptic path names.

📊

Token & Cost Tracking

Monitor input/output tokens and API usage costs for each session in real-time.

🔔

Smart Notifications

Get notified on completion, approval requests, errors, and quota limits. Tap to open the project in your editor.

⏳

Approval Detection

Instantly alerts you when an agent is waiting for your approval to proceed.

📈

Statistics Dashboard

Daily token summary, 7-day usage chart, and top project rankings at a glance.

⌨️

Global Hotkey

Toggle the session panel from anywhere with a fully customizable keyboard shortcut.

🕹️

Pixel Agents Window

Visualize active Claude Code sessions as animated pixel art characters in a floating macOS window. Each character reflects session status — working, waiting for approval, idle — and you can assign a unique pixel character per agent.

🎨

Custom Emoji

Assign unique emoji icons to each project. The menu bar shows a status indicator (▶ …) alongside your emoji so you can tell session state at a glance.

🌙

Quiet Hours

Automatically suppress all notifications during a set time window — works independently from macOS Focus mode.

🚀

Editor Integration

Open projects directly in VSCode, Cursor, Antigravity, IntelliJ IDEA, Terminal, or Finder with one click.

🌐

Bilingual UI

Full Korean and English support — switch languages instantly in Settings.

✨

Auto Update

Stay up to date automatically with built-in Sparkle update support.

🔑

Launch at Login

Auto-start with macOS so your sessions are always monitored from the moment you log in.
